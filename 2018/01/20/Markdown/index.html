前言

  • Windows下可使用VsCode进行Markdown的编辑

    最近打算开始写blog，于是先总结一下Markdown的用法。

介绍

Markdown是一种标记语言，通过简单的标记语法使呈现结果具有更好的可观性，诸多网站，例如github、stackoverflow，都支持Markdown的语法编辑。
另外，现在许多Markdown的编辑器为Markdown的衍生版，扩展了Markdown的功能，如内嵌Html。
Markdown文件的后缀为 .md ，通过Markdown文件可以生成Html文件。
